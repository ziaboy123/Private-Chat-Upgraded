What is this?

This is a private, real-time chat application. It allows you to create temporary, anonymous chat rooms to talk with others.

How it works:

  • Create a Room: Click "Create a Room" on the home page. You will be given a unique 4-letter code and taken directly to your new chat room.
  • Share the Code: Share this code with anyone you want to chat with.
  • Join a Room: Others can enter the 4-letter code on the home page and click "Join a Room" to enter your chat.
  • Anonymous & Private: You only need to pick a name. No other data is stored.
  • Temporary: Chat rooms and all their messages are deleted from memory as soon as the last person leaves the room. Nothing is saved permanently.
